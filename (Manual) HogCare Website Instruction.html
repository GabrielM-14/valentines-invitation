1. Introduction

Purpose of the Manual

The purpose of this instructional manual is to present a thorough discussion of how the website, HogCare, works. Whether you are a new user or someone seeking a refresher, this manual is intended to help you navigate through the pages of the website, may it be to navigate or to utilize a feature in a website, manage your account, and more.

Overview of the Website

HogCare, is a website created specifically for people who are in the pig industry. It aims to help those new and people with experience in their business in hog farming.

2. Getting Started

Accessing the Website

To access HogCare, open your web browser and navigate tohttps://hoggrowingmanagement.infinityfreeapp.com/

Creating an Account

  • create an account:
  • Go tohttps://hoggrowingmanagement.infinityfreeapp.com/login.php
  • Click on [Sign Up].
  • Fill out the required information (Username, First name, Last Name, Password, etc.).
  • Click [Submit].

Logging In

  • log into your account:
  • Go tohttps://hoggrowingmanagement.infinityfreeapp.com/login.php
  • Enter your username and password.
  • Click [Log In].
  • Navigating the Website

Main Menu

In the website, the main menu is located at the top of the page and includes links to Home, Guides, About, Contacts, Login.

Home

The Homepage contains the description of the web application.

Guides

The Guides section contains a swine raising guide and other relevant information.

About

The About section states why the web application should be used by individuals in the pig raising industry.

Contacts

The Contacts section has the information for contacting the developer of the website.

Login

The Login section is where the user may access the website’s functionality further by logging in.

4. Account Types

Worker

In this account type, it allows the user to access the website’s features and use in creating and modifying data of pigs stored in the website’s database.

Manager

In this account type, it provides the user elevated privileges compared to those with worker type accounts such as retrieving multiple pig data from different workers, managing different users classified as Worker, etc.

Administrator

  • this account type, the user has complete control of the system. The administrator has the controls similar to those with manager type accounts and also has control on all of the accounts, being able to modify the user account, enable or disable specific accounts, change specific user’s passwords, etc.
  • Features

The features of the web application are listed below. The feature is only accessible to the account types indicated next to it.

Dashboard (Worker, Manager, Admin)

The dashboard has a summarized data from the various components of the web application such as the pig quantity, feed quantity, income, expenses and pig information.

Pig Profile (Worker, Manager, Admin)

The user can use this feature to create a new pig profile. The following details are requested: Pig ID, Gender, Breed, Date of Birth, Weight (in kg), Length (in mm), Width (in mm), Height (in mm), Notable Physical Features, Feed, Vaccination Report, Disease History, Legality, Farrowing, Farrowing Starting Date, Grade, and Pig Status.

Update Pig Profile (Worker, Manager, Admin)

The user can select a pig to change its information in the database. To access this page, the user must click [Go to Update Pig Profile]. The following details can be changed: Pig ID, Gender, Breed, Date of Birth, Weight (in kg), Length (in mm), Width (in mm), Height (in mm), Notable Physical Features, Feed, Vaccination Report, Disease History, Legality, Farrowing, Farrowing Starting Date, Grade, and Pig Status.

Inventory (Worker & Admin)

The Inventory states how many kilograms or pounds of feed are available for consumption. The values can also be updated by the user. The user can also update the inventory by clicking [Update Inventory]

Cash Flow (Worker & Admin)

The Cash Flow tracks the income and expenses of the farm. Each entry lists the ID, date, item, price and type of transaction.

Farrowing (Worker & Admin)

The Farrowing section displays farrowing sows in the farm. It shows data such as Pig ID, Food, Gender, Breed, Weight, Farrowing Days, and Farrowing Stage.

Guide (Worker, Manager, Admin)

The Guide contains a swine raising guide and other relevant information.

Backup (Worker, Manager, Admin)

The Backup page enables the user to generate a report which includes the summarized information of a specific pig. Managers and Admins can generate a report for all pigs.

Settings (Worker, Manager, Admin)

In the settings page, the user can change his name and password.

Update User Profile (Admin)

The Admin can select a user to change their information in the database. The following details can be changed: ID, Account Type, Name of User, First Name, Last Name, and Account Status.
